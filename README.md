## Telegram Affiliate Product Bot

Python-based Telegram bot that automatically promotes products using affiliate links.
Designed for marketing automation, testing product interest, and driving traffic via Telegram channels or chats.

### Features
- Random product selection
- Affiliate link distribution
- Telegram bot integration (aiogram)
- Lightweight and easy to extend

### Use Cases
- Affiliate marketing automation
- Product promotion in Telegram channels
- A/B testing of product links
- Marketing experiments

### Tech Stack
- Python 3
- aiogram
- AsyncIO

### Security Notice
Bot token and sensitive configuration must be stored in environment variables.
Never commit API keys or tokens to the repository.
